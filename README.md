📊 A complete data preprocessing and exploratory data analysis (EDA) workflow using **R programming**.

This project demonstrates how raw data is transformed into a clean, analysis-ready dataset using standard data science techniques.

---

## 🧠 Workflow Overview

✔ Handled missing values using **mean & mode imputation**  
✔ Detected and treated outliers using **IQR method**  
✔ Cleaned and standardized categorical variables  
✔ Applied data normalization techniques  
✔ Performed feature encoding for machine learning readiness  
✔ Created visualizations using **ggplot2**  
✔ Balanced dataset and split into train/test sets  
✔ Generated statistical summaries for key features
